# 📊 Viva Calif E-Commerce Sales Dashboard | Excel Data Analytics Project

An interactive Excel dashboard project built to analyze and visualize e-commerce sales data for the fashion brand **Viva Calif**.

This project focuses on transforming raw transactional data into meaningful business insights using Excel’s data analytics and visualization capabilities.

---

# 🚀 Project Overview

The objective of this project was to:
- Clean and transform raw e-commerce data
- Analyze customer purchasing behavior
- Identify top-performing products and channels
- Build an interactive dashboard for business decision-making
- Practice data storytelling through visualization

The dashboard provides insights into:
- Revenue performance
- Customer behavior
- Product trends
- Sales channels
- Regional performance
- Inventory distribution

---

# 📌 Key Business Metrics

| KPI | Value |
|------|------|
| 💰 Total Revenue | $649K |
| 📦 Total Orders | 2,400 |
| 👕 Total Quantity Sold | 11,997 |
| ⭐ Average Customer Rating | 4.0 |

---

# 📈 Key Insights

- 📱 Mobile App drives the highest purchases (**36%**)
- 🌐 Website contributes **24%** of orders
- 👟 Sneakers, Jeans, and T-Shirts are the top-selling products
- 👩 Female customers contribute the highest purchases (**52%**)
- 📍 California generates the highest revenue
- 🛒 Customers purchase multiple items per order, indicating strong cross-selling potential
- ⭐ Overall customer satisfaction rating is **4.0**
